# Angular

## 概要
覚書である。順次記載予定。

### 準備
npm install -g @angular/cli

```
Angular CLI: 7.3.8
```

### heroes
ng new angular-tour-of-heroes  
cd angular-tour-of-heroes  
ng serve --open  
http://localhost:4200  
ng generate component heroes  

### deploy
ng build --prod
